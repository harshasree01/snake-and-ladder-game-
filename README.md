# snake-and-ladder-game-

A simple console-based Snake and Ladder game built using Python. This project is beginner-friendly and demonstrates the use of loops, conditionals, functions, lists/dictionaries, and random numbers.


GAME FEATURES;

Single-player or two-player support (turn-based)

Random dice roll (1–6)

Snakes that move the player down

Ladders that move the player up

Win condition at position 100

Clear and simple console output



LANGUAGES USED:

Language: Python 3

Libraries: random (built-)

GAME RULES:

Players start at position 0.

On each turn, a dice is rolled (1–6).

If a player lands on:

Snake head → moves down to the snake’s tail

Ladder bottom → climbs up to the ladder’s top


The first player to reach exactly 100 wins.

If the dice roll exceeds the remaining steps to 100, the move is skipped.
